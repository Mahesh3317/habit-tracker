# Feature Documentation - Self Control & Habit Tracker

Detailed documentation for all app features.

## 🏠 1. Dashboard

### Overview
The main hub showing your overall progress and streaks at a glance.

### What You See
- **Total Points:** Cumulative points from all completed tasks
- **Weekly Progress:** Percentage of days with goals met (0-100%)
- **Streaks:** Current streak for each habit
- **Today's Routine:** Custom task completion progress (if you have custom tasks)
- **Badges:** Achievement unlocks (3-day, 7-day, etc.)

### How to Use
1. Open the app (default screen)
2. Scroll down to see all information
3. Streaks refresh daily based on logging
4. Points accumulate as you complete tasks
5. View today's routine progress at a glance

### Key Metrics
- **Progress Circle:** Shows completion percentage
- **Streak Cards:** Color-coded by habit type
- **Task Progress:** Shows X/Y tasks completed
- **Achievement Badges:** Visual indicators of milestones

### Tips
- Check daily for motivation
- Use task progress to stay focused
- Streaks are your best motivator
- Badges offer long-term motivation

## 📝 2. Daily Tracker

### Overview
Log your daily habits here. This is the most important screen!

### Standard Habits to Log
1. **Cigarettes (Number Input)**
   - Range: 0-50
   - Target: 0 (ideally)
   - Gradual reduction is okay
   - Use +/- buttons or type directly

2. **Private Activity (Yes/No Toggle)**
   - Success: Toggle ON (✓)
   - Failure: Toggle OFF (blank)
   - Binary - no middle ground
   - Completed indicator shows status

3. **Sleep Hours (Decimal Input)**
   - Range: 4-12 hours
   - Target: 7-9 hours
   - Enter like: 7.5, 8, 6.25
   - Critical for self-control!

4. **Workout/Exercise (Yes/No Toggle)**
   - Any physical activity counts
   - 20+ minutes recommended
   - Can be gym, walk, sports, etc.

### Custom Tasks (NEW FEATURE)
- **Create your own daily/weekly tasks**
- **Tap to mark as complete** - Checkmark toggles
- **Automatic reset**: Daily tasks reset at midnight, weekly tasks reset on their day
- **Earn points** for each custom task completed
- **Track progress** with visual indicators
- **See all custom tasks** you've created in your routine

### Completion Check
- Real-time validation
- Shows if each habit meets target
- Visual checkmarks when complete
- "Not logged" shows pending items

### Best Practices
- Log immediately after each activity
- Don't wait until end of day
- Be honest with yourself
- Consult the "Goal badge" for targets
- Create meaningful custom tasks

### Data Persistence
- Automatically saved to phone storage
- Survives app restarts
- Cannot be edited after day ends (future feature)

## ⚡ 3. Urge Control (Emergency SOS)

### Overview
Your emergency button for when addiction triggers hit hard.

### When to Use
- Immediate overwhelming urge
- Temptation becoming unbearable
- Feeling weak or vulnerable
- Stress or emotional triggers

### How It Works

#### Step 1: Start Timer
- Tap big red button
- 5-minute countdown begins
- Motivation quote displays

#### Step 2: Ride the Wave
- Distract yourself for 5 minutes
- Urge typically peaks at 5-15 minutes
- Just survive these minutes!

#### Step 3: Try an Alternative
Pick from 8 alternatives:
- 🚶 **Walk** (5-20 min) - Move your body
- 💧 **Drink Water** (1 min) - Hydrate
- 💪 **Pushups** (5-10 min) - Physical release
- 🧘 **Meditate** (5-10 min) - Calm yourself
- 🚿 **Cold Shower** (5 min) - Reset nervous system
- 🤸 **Stretch** (5-10 min) - Release tension
- 🎵 **Listen to Music** (5-10 min) - Emotional shift
- 📞 **Call a Friend** (10-20 min) - Social support

#### Step 4: Celebrate Victory
- Timer completes
- Motivational message
- You survived! Streak stays intact

### Features
- **Breathing Technique:** 4-7-8 breathing guide
- **Cold Shock:** Scientific explanation
- **Streak Display:** Shows what you're protecting
- **Multiple Quotes:** Different motivation each time

### Science Behind It
- Urges have a natural lifespan (5-15 min)
- Distraction disrupts the addictive loop
- Physical activity resets dopamine pathways
- Cold exposure activates parasympathetic nervous system

### Pro Tips
- Use immediately (don't negotiate)
- Combine urge control with alternative action
- Pre-plan your alternatives beforehand
- Track how many times you use it (future feature)

## 🎯 5. Custom Task Management (NEW)

### Overview
Create and manage your own personalized daily or weekly tasks. Go beyond the default 4 tasks!

### Creating a Custom Task

#### Task Details
- **Task Name** (required, max 50 chars)
  - Examples: "Read 30 minutes", "Call mom", "Journal", "Meditate"
- **Description** (optional, max 200 chars)
  - Add context and motivation
- **Repeat Type**
  - **Daily**: Task appears every single day
  - **Weekly**: Task appears only on the day you created it

#### Advanced Settings
- **Reminder** (optional)
  - Enable notifications at a specific time
  - Default: 9:00 AM if enabled
  - Customize the time
  - Get reminded to complete the task

- **Points Reward** (5-100 points)
  - Slider to set reward value
  - Default: 10 points
  - Higher difficulty = more points
  - Motivate yourself with meaningful rewards

### Managing Your Tasks

#### Task List View
- See all created custom tasks
- Filter by day of week for weekly tasks
- Show today's tasks count
- Display active vs inactive tasks

#### Task Actions
- **Edit Task**: Modify name, description, repeat type, reminder, or points
- **Toggle Active**: Temporarily disable without deleting
- **Delete Task**: Remove permanently (with confirmation)
- **View History**: See last 10 completions

### Task Completion
- **Mark Complete**: Tap task in Daily Tracker to check it off
- **Auto-Feedback**: See point increase immediately
- **Today's Progress**: Track completion percentage in Dashboard
- **Automatic Reset**: 
  - Daily tasks reset at midnight (00:00)
  - Weekly tasks reset on their designated day

### Best Practices for Custom Tasks
1. Start with 1-2 custom tasks (don't overload)
2. Use reasonable point values (higher = more valuable)
3. Enable reminders for habits you forget
4. Weekly tasks are great for irregular activities
5. Review and adjust tasks monthly

### Examples of Good Custom Tasks
- "Drink 8 glasses of water" (daily)
- "Cold shower" (daily)
- "Read 30 minutes" (daily)
- "Clean room" (weekly)
- "Call accountability partner" (weekly)
- "Review goals and progress" (weekly)

## 📊 6. Task Analytics (NEW)

### Overview
Deep insights into your custom task completion patterns and performance trends.

### Overall Statistics
- **Total Tasks Created**: Shows how many custom tasks you've built
- **Average Completion Rate**: Overall performance across all tasks
- **Most Consistent Task**: The task you complete most often 🌟
- **Needs Focus**: Task with lowest completion rate ⚠️
- **Tasks at Risk**: Tasks in 50-99% completion range

### Per-Task Analytics

#### Key Metrics (Last 7 Days)
- **Completion %**: How often you completed this task (0-100%)
- **Current Streak**: Consecutive days/weeks completed
- **Best Streak**: Your personal record for this task
- **Visual Chart**: Last 7 days completion pattern
  - Bar height = completed
  - Color coded by performance

#### Performance Levels
- 🟢 **Excellent** (100%): Perfect execution
- 🔵 **Great** (75-99%): Consistent and reliable
- 🟠 **Good** (50-74%): Building consistency
- 🔴 **At Risk** (1-49%): Needs attention
- ⚫ **Not Started** (0%): No data yet

#### Smart Insights
- Auto-generated feedback
- Personalized recommendations
- Motivational messages
- Specific to each task

### Using Analytics Effectively
1. **Track Trends**: Look at 7-day patterns, not daily fluctuations
2. **Identify Weak Areas**: Focus on red/orange tasks
3. **Celebrate Wins**: Blue/green tasks are successes!
4. **Adjust Difficulty**: Lower points if task is too hard
5. **Build Streaks**: Consistency matters more than perfection

### Example Insights
- "Perfect! Completing [Task] 100% of the time 🔥"
- "Great job! [Task] at 85% - Keep the momentum!"
- "Get started with [Task] - Track your first completion!"
- "You haven't completed [Task] in 5 days - Time to refocus!"

## 📱 12. Technical Details

### Data Persistence
- **Habits:** Local AsyncStorage on device
- **Custom Tasks:** Stored separately with completion logs
- **Auto-saves:** Every change saved automatically
- **Survives:** App restarts
- **Lost if:** You delete app data

### Notifications
- Expo Notifications API
- Scheduled daily (6AM, 6PM, 10PM)
- Custom reminder times for tasks
- Requires permission

### State Management
- React Context API with useReducer
- Centralized habit + task state
- Automatic AsyncStorage persistence
- Seamless updates across screens

### Performance
- Optimized for mobile
- Minimal dependencies
- Smooth animations
- Fast data access
- Efficient task analytics calculations
