# 🚀 QUICK START - 3 Steps to Success

## 1️⃣ Hot Reload on Mobile (Development)

```bash
npm start
```

**Output in terminal:**
```
▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄
█ ██ █  Expo █
█ ██ █████████

› Metro waiting on exp://YOUR_IP:8083
› Scan the QR code above with Expo Go
```

**On Your Phone:**
- Install [Expo Go](https://expo.dev/tools/expo-go)
- Scan QR code with phone camera
- Watch app open instantly!
- Edit code → Save → Auto-reload ⚡

**Press in terminal:**
- `w` = Open in web browser
- `r` = Reload app
- `?` = Show all commands

## 🆘 Errors & Fixes

### "Port already in use"
```bash
npm start  # Just use different port when prompted
```

### "Can't connect to app"
- Same WiFi network? ✅
- Restart dev server: `Ctrl+C` then `npm start`
- Restart Expo Go app

### "App crashes"
- Check error message on screen
- Look at terminal for details
- Fix code and press 'r' to reload

### "Build quota exceeded"
- Free plan: 1 build/month
- Wait until next month, or
- Upgrade Expo plan for unlimited builds
